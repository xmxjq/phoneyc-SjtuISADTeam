Requirements:
=============

    libemu-svn
    http://libemu.carnivore.it/
    
    curl
    http://curl.haxx.se/


Compile the modules:
====================

    $ cd modules
    $ make
    $ make install

NOTE: Don't need to have root privilege when running make install.


Run it:
=======

   $ python phoneyc.py URL-you-what-to-examine
